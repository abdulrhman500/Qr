# Qr-Generator
Generates QR-code images from excel sheet and adds QR-code image link to the excel sheet

```bash
git clone https://github.com/abdulrhman500/Qr-Genrator.git
python -m venv .venv
.venv\Scripts\activate
pip install -r "requirements .txt"  
```
